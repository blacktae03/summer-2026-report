Matrix Factorization
Koren et al., 2009 — Netflix Prize 우승팀의 핵심 알고리즘
  • 기본 아이디어 — 유저·아이템을 같은 잠재요인 공간에 매핑, 내적으로 평점을 예측
  • 확장 ① — Bias(개인 평점 성향), 암시적 피드백, 인구통계 정보 결합
  • 확장 ② — Temporal Dynamics: 취향은 고정이 아니라 시간에 따라 변함(Power-law 감쇠)
  • 확장 ③ — Confidence: 어뷰징·일회성 클릭보다 반복 행동에 더 큰 가중치
INSIGHT
차원 수(f)를 200으로 늘린 단순 모델보다, Bias·암시적 피드백 같은 정교한 파라미터를 추가한 모델이 차원 10만으로도 오차가 훨씬 낮았다.

Factorization Machines
Steffen Rendle, 2010 — 희소 데이터에서 SVM의 한계를 넘다
문제
유저·아이템을 원-핫 인코딩하면 두 변수가 같이 등장한 적 없는 조합의 상호작용 파라미터는 학습 자체가 불가능
해결
상호작용 가중치를 독립값 대신 두 잠재벡터의 내적으로 분해 — 서로 만난 적 없어도 벡터로 간접 추정
효율
완전제곱식 트릭으로 연산량을 O(kn²)에서 O(kn)까지 압축 — 1억 건 규모에서도 학습 가능

Neural Collaborative Filtering · SVM Duality
NCF (He et al., 2017)
  • 내적이라는 선형 연산으로 상호작용 함수를 고정한 게 MF의 한계
  • Jaccard 계수를 반례로 사용해 내적의 기하학적 제약(삼각부등식)을 증명
  • 내적 대신 MLP로 비선형 상호작용을 직접 학습
SVM Duality (CS229)
  • Primal ↔ Dual: 최소·최대의 연산 순서를 바꾸는 것뿐
  • Convex + Affine 조건 → 강쌍대성 → KKT → Support Vector
  • 새 데이터는 소수의 Support Vector와의 내적만으로 분류 — Kernel Trick의 토대

문제 정의 — HUG는 어떻게 돈을 버는가
  • HUG는 보험회사다 — 사고 안 난 다수의 보증료로 사고 난 소수의 손실을 메움
  • 관건은 회수율이 아니라 사고율 — 보증료율은 정책상 낮게 고정, 마진이 매우 얇음
  • 2023년 대위변제 4.9조 폭증 → 보증료 수입의 5배 초과 → 순손실 −3.9조
  • 공적 사업이라 가격(요율) 레버는 못 씀 → 남은 레버는 사고율↓ · 회수율↑ 뿐
데이터
전세보증 대위변제 66,580건, 경매현황 32,543건 등 8개 표 데이터 — 전부 10만 행 이하 tabular
